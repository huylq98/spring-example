# spring-example

##1. Tech Stack
* Frameworks: Spring Boot, Spring JPA.
* Cache: Redis.
* Database: H2.
* Build tool: Maven.
* Techniques:
    * Using `@RestControllerAdvice` to handle exceptions globally.
    * Using Mapstruct to convert between models.
    * Using `Filter` and `@ControllerAdvice` to log requests and responses.
